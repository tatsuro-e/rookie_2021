研修内容

内部研修

  • ビジネスマナー研修
    • 社会人としての基礎的な能力を養うための研修。 主に文書作成術や、社会人としてのマナー、タイムマネジメントなどについて学ぶ。
  • セキュリティ教育
    • 情報機器の使い方、情報管理など、情報セキュリティについて学ぶ。

外部研修

  • アルゴリズム
    • アルゴリズムの概要を学び、チームに分かれて演習を行う。
  • IT基礎
    • 基本的なIT知識について学び、疑問点などの確認をグループディスカッションを通じて行う。
  • Linux
    • OSの一種であるLinuxの概要と簡単なコマンド操作を学ぶ。
  • Git
    • バージョン管理ツールであるGitの概要とGitLabの操作を学ぶ。
クライアントサイド
  • HTML/CSS
    • 概要を学び、自己紹介ページを作成。HTML/CSSを学ぶことで、Webページの表示ができるようになる。
  • JavaScript
    • JavaScriptの文法と書き方を学ぶ。JavaScriptを使用することで、Webページに動きをつけることができるようになる。
  • React/Material-UI
  • SPA開発演習
    • クライアントサイドで学んだことを活用して、簡易的なWebアプリケーションの開発（SNS、投票サイトなど）を行う。
サーバーサイド
  • Java
    • Javaの文法と書き方を学ぶ。Javaを学ぶことで、サーバサイドの構築ができるようになる。
  • データベース
    • データベースの概要を学び、MySQLを使用してデータベースを構築する。
  • Spring Boot
    • Spring Bootの概要を学び、データベースアクセスの演習を行う。
  • SPA+REST開発演習
    • Webアプリケーションのサーバ―サイドの開発演習を行う。
システム構築演習
  • 詳細設計
    • システムのデータベースやデータベースとのやり取りについて設計を行う。
  • テスト設計
    • システムに想定されるバグを予測し、テスト項目を洗い出す。
  • 実装・単体テスト
    • 設計書を基に、プログラミングを行う。プログラミングしたものは、動作確認を行う。
  • テスト
    • テスト設計書を基にテストを行い、見つかったバグを修正する。
